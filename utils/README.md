# utils

业务无关工具库

## 目录结构说明

```
utils
  |--src/main/java/com/example/utils //多级目录折叠
       |--anim //生成动画工具类
       |--permission //运行时申请权限模块
       |--ClipboardUtil //复制粘贴文本的三种类型
       |--DensityUtil //尺寸大小转换工具类：dp与px、sp与px
       |--DeviceUtil //设备相关工具类：屏幕大小、状态栏等
       |--LogUtil //打印日志工具类
       |--StatusBarUtil //控制状态栏的工具类
```
